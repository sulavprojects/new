Font installation Guide on windows

  • Darkseid
  • Jan 1, 2020

Installing fonts on windows Os is simple process follow the installation guide below.

At first download the font you want to install from "arabfonts.org"

Fonts installation process.

  • extract or unzip the download file of fonts
  • right click on file you want to install (it should be in TTF format) then options will appear
  • click on install for all user
  • now you are good to go
  • Try install fonts appear on Ms office or not
